主要作用: 用来测试模块运行消耗的内存占用和耗电量。
备注: 使用的是坐标点击，不同的分辨率需要重写坐标值

注意点: 当前文件夹下面的文件和文件夹要慎重删除。

各个文件夹的作用: 
1，apk文件夹：用来存放需要测试的APP包，要测试的包名最好全是英文，不要存在特殊字符，以免报错。
2，case文件夹：用来存放需要操作的配置文件，需要根据手机分辨率填写(已存在的可在1080x1920手机上面运行)
	case.txt文件内容含义:
	        click(905,1421，10):表示点击，参数 :clickX clickY sleeptime
			home(10):表示点击home
			back(10):表示点击back
			leftslide(978,885,80,885,10):表示向左滑动，参数 : fromX1 fromY1 toX2 toY2 time
			rightslide(80,885,978,885，10):表示右滑, 参数 : 同上
			upslide(80,885,978,885，10):表示上滑，参数 : 同上
			downslide(527,176,527,1611，10): 表示下滑，参数 : 同上
			bright_screen(1):表示亮屏，灭屏也是使用这条，参数：同上
			long_swipe(527,176,527,176,1,10):表示长按，4个表示坐标，第5个为点击时间，第6个为等待时间。
			sleep():表示睡眠时间。
			locker_locker():开启locker。
			screen_saver():开启屏保。
3，instructions文件夹：

	此文件夹用来放置生成的数据信息。当运行py文件以后，会自动生成文件放置在里面。
	
4，运行方法：
			1，首先我们需要将case.txt和splash.txt配置文件里面的内容按照测试机器填写一下测试信息。
			
			2，将需要测试的apk包放在apk文件夹下面。
			3，在当前文件夹下面执行命令行操作，将modelMeminfo.py文件拖到命令行中，后面空一格然后添加测试次数，然后就可以执行了。
